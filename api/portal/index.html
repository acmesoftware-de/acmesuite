ACMEmailtrap

Mail trap for testing

Separate product: catches outgoing mail from ACMEsuite over SMTP, stores it per recipient and serves it via IMAP and this REST API. Open source.

Self-contained specs (bundled) for download: hr · crm · supply · build · all  |  shared building blocks: acme-base  |  separate product: acme-mailtrap

Source of truth: acmesoftware-de/acmesuite · api/. Auth: bearerAuth (JWT), roles WATCH·WORK·ADMIN. Errors per RFC 7807.
